Checklist questions for Desktop

  • In systems that use overlapping windows, is it easy for users to rearrange windows on the screen?
    YES NO N/A
  • If setting up windows is a low-frequency task, is it particularly easy to remember?
    YES NO N/A
  • Are function keys labeled clearly and distinctively, even if this means breaking consistency rules?
    YES NO N/A
  • Do on-line instructions appear in a consistent location across screens?
    YES NO N/A
  • If pop-up windows are used to display error messages, do they allow the user to see the field in error?
    YES NO N/A
  • Provide a text equivalent for every non-text element
    YES NO N/A
  • Ensure that all information conveyed with color is also available without color, for example from context or markup
    YES NO N/A
  • Clearly identify changes in the natural language of a document's text and any text equivalents
    YES NO N/A
  • Ensure that foreground and background color combinations provide sufficient contrast when viewed by someone having color deficits or when viewed on a black and white screen
    YES NO N/A
  • If, after best efforts, you cannot create an accessible page, provide a link to an alternative page that uses W3C technologies, is accessible, has equivalent information (or functionality), and is updated as often as the inaccessible (original) page
    YES NO N/A
  • Until user agents allow users to control blinking, avoid causing content to blink
    YES NO N/A
  • Until user agents provide the ability to stop auto-redirect, do not use markup to redirect pages automatically. Instead, configure the server to perform redirects
    YES NO N/A
  • In multipage data entry screens, is each page labeled to show its relation to others?
    YES NO N/A
  • Is there visual feedback when objects are selected or moved?
    YES NO N/A
  • Is there feedback when function keys are pressed?
    YES NO N/A
  • If there are observable delays (greater than fifteen seconds) in the system’s response time, is the user kept informed of the system's progress?
    YES NO N/A
  • Typing, cursor motion, mouse selection: 50-1 50 milliseconds
    YES NO N/A
  • When prompts imply a necessary action, are the words in the message consistent with that action?
    YES NO N/A
  • Does the system automatically enter a dollar sign and decimal for monetary entries?
    YES NO N/A
  • Has the system been designed so that keys with similar names do not perform opposite (and potentially dangerous) actions?
    YES NO N/A
